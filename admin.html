A Definition of the Admin

The Admin SDK Directory API lets administrators of enterprise domains view and manage their organization's users, groups, devices, and related resources. To call this service, we recommend that you use the Google-provided client libraries. If your application needs to use your own libraries to call this service, use the following information when you make the API requests.
